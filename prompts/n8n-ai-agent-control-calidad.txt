```sql
  COUNT(*) FILTER (
    WHERE UPPER(TRIM(COALESCE(se_detecto_situacion_critica,''))) = 'SI'
      OR UPPER(TRIM(COALESCE(actitud,''))) IN ('MAL','REGULAR','DEFICIENTE')
      OR UPPER(TRIM(COALESCE(presencia,''))) IN ('MAL','REGULAR','DEFICIENTE')
      OR TRIM(COALESCE(observaciones_detallar, detallar_situacion_detectada, '')) <> ''
  ) AS resultados_negativos
FROM supervisiones_formulario_paciente_prestador_respuestas
WHERE marca_temporal >= '[inicio]' AND marca_temporal < '[fin]';
```
Usar Query C para los totales de las dos secciones. resultados_positivos + resultados_negativos puede ser < total si hay visitas neutras; clasificar neutras como positivas solo si no hay incidencia.

### Columna "Resultado" en Anexo

2. Si hay observaciones relevantes → **Observación operativa** con texto concreto de Query B.
3. Si no → resumir evaluación: actitud, presencia, vínculo paciente, higiene (solo campos con valor).
4. Patología/movilidad/estado anímico → solo en **Resultado** o narrativa, NO en columnas de nombres.
